# FP. Asset Manifest

Registry of all approved brand assets. Managed by fp-creative-direction when assets are approved through fp-design-lab or created directly.

## How This Works

Every approved asset gets an entry here with:
- Filename and location
- Creation/approval date
- Source (original, adaptation, photography)
- Usage tags (where it can be used)
- Brief description

## Usage Tags Reference

- `social-linkedin` ... LinkedIn posts, carousels
- `social-general` ... Cross-platform social
- `web-hero` ... Website hero sections
- `web-section` ... Website content sections
- `blog-featured` ... Blog featured images
- `email-header` ... Newsletter headers
- `email-body` ... Newsletter body imagery
- `presentation` ... Slide decks
- `avatar-profile` ... Profile pictures, avatars
- `banner` ... LinkedIn banner, website banners
- `video-thumbnail` ... YouTube/video thumbnails
