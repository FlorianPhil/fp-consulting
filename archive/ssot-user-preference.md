SSOT (Single Source of Truth) is a foundational principle for all work, all projects. Every piece of content, data, and asset has exactly one authoritative source. If something exists in two places, one is the source and the other is a synced copy. Before creating or editing anything, identify the SSOT first. After changes to any deployed copy, sync back to the source. If a discrepancy is found, flag it and ask which version is correct. Never duplicate without a sync mechanism. When in doubt about which version is current, the SSOT wins.
